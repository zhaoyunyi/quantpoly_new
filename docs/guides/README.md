# 协作索引（docs/guides）

本目录承接根 `AGENTS.md` 与 `CLAUDE.md` 里重复的项目级协作背景，避免相同说明在多个入口文档中重复维护。

## 文档列表

- `ai-collaboration.md`：仓库级协作入口、硬约束、常用命令、目录入口
- `ai-context-index.md`：按领域和文档类型查找上下文的索引地图

## 使用方式

建议先读：

1. `README.md`
2. `docs/guides/ai-collaboration.md`
3. `docs/guides/ai-context-index.md`
4. `openspec/AGENTS.md`（涉及计划 / 提案 / 规格变更时）
5. 当前工作目录下的 `AGENTS.md` / `CLAUDE.md`
